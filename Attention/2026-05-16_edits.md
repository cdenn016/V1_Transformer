### Deleted: `aux_layer_loss` and `aux_loss_weight`

The per-layer auxiliary CE loss for non-final layers is removed. Per
CLAUDE.md the architecture is supposed to be "no neural networks" — the
auxiliary M-step task signal on intermediate layers added a non-canonical
gradient path that does not appear in the manuscript free-energy
functional. Default value was `False` everywhere, so no production run
loses behavior.

### Deleted: `hierarchical_priors`

The boolean flag controlling whether each layer's posterior μ becomes
the next layer's prior μ is removed, and the **cross-layer μ cascade
is fully removed** along with it. Every layer now receives the same
embedding-derived `mu_prior` for the entire stack — matching the
behavior `train_publication.py` and `scripts/run_ablation_suite.py`
were already running (`hierarchical_priors=False`). The dataclass
default was `True`, but the production configs that mattered
overrode to `False`, so this choice preserves publication metrics on
re-run. `scripts/run_optuna_hpo.py` and
`scripts/verify_active_inference.py` ran with `True` and will now
match the publication baseline instead — those scripts are research
utilities, not the canonical workload.

The canonical VFE hierarchy `h → s → p → q → observations` and the
`lambda_hyper · KL(s_i || h)` term in the free-energy formula are
unrelated to this flag — the verifier traced 39 `lambda_hyper`
references and confirmed independence.

Surgery sites:

- `transformer/core/block_config.py`: dataclass field, `from_config`
  read, and the inline comment listing it as a key toggle — all removed.
- `transformer/core/blocks.py`: instance attribute, gradient-checkpointing
  capture-by-value comment, and the entire μ-cascade block (both the
  outer `if self.hierarchical_priors` gate and the inner
  `_em_active` / `amortized_inference` detach branches). `mu_prior`

- `transformer/core/model.py`: entire μ-cascade block removed

- `transformer/config_values_for_settings.py`: dict default removed.
- Scripts and `transformer/train_publication.py`: dict entries removed.
