## Issue 2b: Factor 1/2 Dropped Incorrectly (Supplement)

## Issue 3: Dirac KL Not Literally Valid (Main Text)

**Problem:** Lines 1292-1299 wrote $D_{\mathrm{KL}}(\delta(\mu_i) \| \mathcal{N}(\mu_p, \sigma_p^2 I)) = \frac{1}{2\sigma_p^2}\|\mu_i - \mu_p\|^2$, but KL between a Dirac and a Gaussian is literally $+\infty$.

## Logit Kernel Factorization (lines 1148-1152, 1627)

**Problem A:** Line 1150 identified "the effective bilinear form $M^a = (W_Q^a)^\top W_K^a \in \mathbb{R}^{d_\text{head} \times d_\text{head}}$" as the logit kernel. That is not the matrix appearing in standard attention logits. The actual logit is $x_i W_Q^a (W_K^a)^\top x_j^\top$, so the ambient-space kernel is $W_Q^a (W_K^a)^\top \in \mathbb{R}^{d_\text{model} \times d_\text{model}}$ with rank $\leq d_\text{head}$ — a low-rank matrix, not an element of $GL(d_\text{model})$.

**Problem B:** Line 1152 stated the rectangular shape "encodes both the orthogonal projection onto block $a$ and the GL(d_head) transport." Too strong for a generic learned $W_Q^a$, which is not literally an orthogonal projector onto a fixed canonical block.

**Fix:** Rewrote lines 1148-1152 to use the thin SVD factorization: $W_Q^a = U_Q^a A_Q^a$ where $U_Q^a$ has orthonormal columns (isometric subspace embedding) and $A_Q^a \in GL(d_\text{head})$ is the invertible head-space map. The gauge identification $\sigma^{-2}\Omega^{-\top} \leftrightarrow M_h^a = A_Q^a (A_K^a)^\top \in GL(d_\text{head})$ operates at the level of the invertible head-space factor, not the ambient low-rank kernel. Added labeled equation `eq:head_space_kernel`. Updated corresponding passage in the multihead section (line 1627) to use the same factorization with cross-reference.

## "Standard Transformers Implement Gauge Structure" Overclaim

**Problem:** The manuscript repeatedly states that "standard transformers already implement gauge structure" or "$W_Q, W_K \in GL(d_k)$." Both are wrong as stated. Standard per-head projections $W_Q^a, W_K^a \in \mathbb{R}^{d_\text{model} \times d_\text{head}}$ are rectangular and not invertible. The manuscript itself acknowledges this at lines 1150-1152 (the identification is within the $d_\text{head}$-dimensional projected subspace) and at line 1622 (rectangular shape = projection + GL(d_head) map). But 6 other locations make the unqualified claim.

Moreover, the identification only holds in the constant-gauge, isotropic, flat-bundle, deterministic limit. Standard transformers do not have transported covariances, explicit $\Omega_{ij}$, non-flat connections, or belief-state dynamics.

**Fix (6 locations):**
- Abstract: "suggesting...already implement gauge-theoretic attention" → "suggesting...can be interpreted as a degenerate case of gauge-theoretic attention operating in this limit"
- Line 566: "$W_Q, W_K \in GL(d_k)$" → corrected to reference the bilinear form, notes rectangular shape, cites Section on multihead
- Line 1607: "reveals this as implementing" → "Under the gauge-theoretic identification, this corresponds to"
- Line 1611: "standard transformers implement $GL(d_k)$ gauge-theoretic attention" → "in the isotropic flat-bundle limit, standard transformer attention can be interpreted as"
- Line 2183: "already implement gauge symmetry" → "can be interpreted as a degenerate case...operating in the constant-gauge, isotropic, flat-bundle limit"
- Conclusion (line 2214): "reveals that standard transformers already implement this gauge structure" → "permits interpreting standard transformer attention as a degenerate case of this gauge structure"

## GL+(K) Surjectivity Error

**Problem:** Two locations (main text line ~1952, supplement line ~552) claimed $\exp: \mathfrak{gl}(K) \to GL^+(K)$ is surjective. This is false: matrices in $GL^+(K)$ with negative real eigenvalues of odd Jordan-block multiplicity have no real logarithm (Culver 1966). The code already correctly documents this (in `generators.py`, `transport.py`, `gauge_utils.py`).

**Fix:** Both locations now state that a single $\exp$ is NOT surjective, but the pairwise transport $\Omega_{ij} = \exp(\phi_i)\exp(-\phi_j)$ covers all of $GL^+(K)$ via polar decomposition: any $A \in GL^+(K)$ factors as $A = PO$ where $P = \exp(\log P)$ (symmetric positive-definite) and $O = \exp(S)$ with $S \in \mathfrak{so}(K)$. Added Culver 1966 citation.

## Session 3: Full-Covariance Renyi alpha-Divergence Implementation

Generalized Renyi alpha-divergence from the diagonal-covariance path to the full-covariance path across 3 files. No new configuration surface was added. This removes the NotImplementedError and two warning+fallback blocks that silently degraded to KL when alpha != 1.0.

### Mathematical basis

For full-covariance Gaussians, the blended covariance is Sigma_blend = (1-alpha)*Sigma_q + alpha*Sigma_t. The divergence is:

    D_alpha(q||p) = 0.5 * [ alpha * delta_mu^T Sigma_blend^{-1} delta_mu
                           + 1/(alpha-1) * ((1-alpha)*log|Sigma_q| + alpha*log|Sigma_t| - log|Sigma_blend|) ]

Self-coupling gradients (verified against the diagonal reference in kl_computation.py lines 312-331 and vfe_gradients.py lines 600-614):

    d D_alpha / d mu_q    = alpha_d * Sigma_blend^{-1} * (mu_q - mu_p)
    d D_alpha / d Sigma_q = 0.5*(Sigma_blend^{-1} - Sigma_q^{-1})
                            - 0.5 * alpha_d * (1-alpha_d) * Sigma_blend^{-1} * delta_mu * delta_mu^T * Sigma_blend^{-1}

Both expressions reduce exactly to the existing diagonal formulas when all covariances are diagonal.
