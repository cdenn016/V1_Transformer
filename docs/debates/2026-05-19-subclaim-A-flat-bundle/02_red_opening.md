# Red Opening — subclaim-A-flat-bundle

## Steelman (opposing position)

Setting `Ω_{ij} = Ω` is a specialization that selects one orbit representative inside the upstream GL(K)-gauge framework: Theorem `thm:glk_invariance` continues to hold globally for all `Ω ∈ GL(K)`, so the downstream "standard attention" form sits inside the gauge-equivariant theory and inherits its upstream invariance — nothing is destroyed, only one configuration is named.

## Position

The flat-bundle limit `Ω_{ij} = Ω` is a **gauge fixing**, not a specialization, and the manuscript's downstream argument confuses two distinct invariances. Theorem `thm:glk_invariance` (`GL(K)_attention.tex:520–555`) is invariance of KL under *simultaneous push-forward of both arguments by a single global `Ω`*; the §5.7 summary at `GL(K)_attention.tex:1978` uses it to justify a *local, per-agent* equivariance claim (`W_Q, W_K` "are valid gauge transformations"). These are different objects. Once `Ω_{ij} = Ω` is fixed, only the diagonal subgroup `g_i = g` stabilizes the configuration; per-agent gauge action with `g_i ≠ g_j` moves it. The "upstream gauge-equivariance argument" the downstream claim invokes is therefore not preserved by the limit — it never applied to the limit object in the first place, because the limit object lives in a fixed gauge.

## Evidence

- **The theorem statement is single-Ω, simultaneous push-forward.** `Attention/GL(K)_attention.tex:520–555` states `D_KL(Ω_* P ‖ Ω_* Q) = D_KL(P ‖ Q)` for a single `Ω ∈ GL(K)` applied to *both* arguments. The proof at lines 539–553 only addresses this case: the `(det Ω)²` cancels because the same `Ω` appears in both numerator and denominator of the log-det ratio. There is no per-agent version derived. This is invariance of KL as a function of two distributions under a global change of variables — a textbook property of any f-divergence (`external_canon_math.md` §1, "KL ... invariant under bijective change of variables"). It is not local gauge equivariance.

- **The downstream invocation is a local equivariance claim.** `Attention/GL(K)_attention.tex:1978`: "the `GL(K)` gauge invariance theorem ... guarantees that the learned projections `W_Q, W_K` are valid gauge transformations, explaining why transformers can learn arbitrary (invertible) attention patterns." This treats each learned matrix as an *independent* `Ω`-valued element acting on its own agent — a per-agent action with `g_i ≠ g_j` in general. Per `external_canon_math.md` §2: "Gauge invariant: `Q(g·s) = Q(s)`. Gauge equivariant: `Q(g·s) = ρ(g) Q(s)` ... Conflating these is a common error." The theorem proves a *single-Ω invariance* property of a scalar KL value; the summary uses it as license for *per-agent equivariance* of a vector-valued attention map. The implication does not run.

- **Constant-Ω is a gauge fixing in the canonical sense.** `Attention/GL(K)_supplementary.tex:53` admits: "Throughout the main paper we work with the vertex-frame parameterization `Ω_{ij} = exp(φ_i)exp(-φ_j)`, which encodes a *globally trivial* principal G-bundle ... and the curvature 2-form of the induced connection vanishes identically (Lemma `vanishing_holonomy` of the main text). The principal-bundle vocabulary developed in this appendix is therefore strictly more general than the parameterization used by the language model and the BERT validation, both of which live in the flat-bundle subclass." The working framework already has curvature `F = 0` and trivial holonomy by construction; the further reduction to `Ω_{ij} = Ω` picks one point on the gauge orbit. Per `external_canon_math.md` §2 ("Curvature = 0 iff connection is flat (locally trivializable as a constant bundle)") and the canon's gauge-fixing summary (Nakahara §10.4 / Frankel Ch. 18 lineage), *fixing the connection to a constant element is gauge fixing*, and a gauge-fixed Lagrangian is not gauge-invariant — gauge invariance is a property of the *unfixed* Lagrangian. The fixed-gauge object only retains the residual stabilizer (here, the diagonal subgroup `g_i = g`), not the full per-agent GL(K) action.

- **The "Ω_{ij} = Ω fixed configuration" is not stabilized by the full gauge group.** Under the per-agent action `Ω_{ij} → g_i Ω_{ij} g_j^{-1}` (the manuscript's own transformation law for connections, e.g. `Attention/GL(K)_attention.tex:510`), the configuration `Ω_{ij} = Ω` for all `(i,j)` maps to `g_i Ω g_j^{-1}`, which equals `Ω` only when `g_i Ω g_j^{-1} = Ω` for all `i,j` — i.e., `g_i = Ω g_j Ω^{-1}` for all pairs, which forces `g_i = g` constant (taking `i = j`). The stabilizer is the diagonal `G`, not the full `G^N`. This is the textbook signature of gauge fixing: the orbit is collapsed to a slice, and only the residual (global) symmetry survives.

- **The natural-gradient argument is auxiliary but worth flagging.** `external_canon_transformers.md` §9: "Killing form: ... Sign-indefinite on non-compact groups; ... **Not positive-definite on gl(K).**" The §5.7 chain (`GL(K)_attention.tex:1962`) bundles "variational natural-gradient descent on the gauge-equivariant free energy" with the flat-bundle limit. Once `Ω` is fixed to a constant and absorbed into `W_Q W_K^T`, the residual "natural gradient" reduces to ordinary preconditioned gradient descent on the W-parameter space, which has no canonical Fisher-Rao structure from the gauge side (no per-agent statistical manifold remains to define a Fisher metric across agents). This is a tertiary point: blue can plausibly answer "we use the Fisher metric on the statistical Gaussian fiber, not the Killing form on gl(K)" — but in the flat-bundle limit even that fiber-Fisher metric is pre-fixed to a constant, so the "natural" qualifier is empty.

## Falsification conditions

This position is wrong if any one of the following holds:

1. The "upstream gauge-equivariance argument" §5.7 invokes is shown to be something other than Theorem `thm:glk_invariance`, *and* that other quantity is established to be preserved under `Ω_{ij} = Ω` without requiring re-derivation in the fixed gauge. (I read the §5.7 text at `GL(K)_attention.tex:1978` to be invoking precisely the simultaneous-Ω theorem; if blue points to a separate per-agent equivariance theorem with its own proof, the conflation charge breaks.)

2. The constant-Ω configuration is shown to be stabilized by the full per-agent gauge action `Ω_{ij} → g_i Ω_{ij} g_j^{-1}` (not merely the diagonal subgroup `g_i = g`). My stabilizer computation above shows otherwise from the manuscript's own transformation law; blue must either contradict that computation or supply a different transformation law.

3. The canonical gauge-theory literature is shown not to treat "fix connection to constant" as gauge fixing — i.e., `[Nakahara2003 §10.4]` or `[Frankel2011 Ch. 18]` actually treat such a specialization as preserving the unfixed gauge invariance of the action functional in the technical sense the manuscript needs.

If none of these breaks, the manuscript's "standard transformer = gauge-equivariant attention" claim describes only the *trace* of upstream invariance left after gauge fixing (the diagonal `G`-stabilizer, equivalent to a global similarity transform on the W-matrices), not the per-agent gauge equivariance the §5.7 summary appears to assert.
