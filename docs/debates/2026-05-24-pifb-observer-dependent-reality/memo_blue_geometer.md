# Memo — geometer (BLUE) — pifb-observer-dependent-reality

## Steelman of the attack
Sub-point 2, sharpened as an internal-consistency charge: it is incoherent to say $\mathcal{C}$ "possesses no intrinsic metric" while the $G_i$ are pulled back from a *fixed* Fisher-Rao fiber metric. If the fiber metric is fixed and the same for all agents, then the geometry is not really agent-relative; the substrate-plus-fixed-fiber-metric is a hidden god's-eye structure, contradicting "no view from nowhere."

## Defense, derived from external canon
Pulling back a fixed fiber-side metric onto a base that carries no metric of its own is geometrically standard and free of contradiction. The pullback of a metric by a map is $G_i = \sigma_i^* g_{\mathcal{B}}$, where $g_{\mathcal{B}}$ is the (fixed) Fisher-Rao metric on the belief fiber $\mathcal{B}$ and $\sigma_i$ is agent $i$'s section [Lee2013, pullback of tensors; doCarmo1992 §1–2]. Two facts resolve the apparent contradiction.

First, the pullback metric on $\mathcal{C}$ depends on the section $\sigma_i$ (the differential $d\sigma_i|_c : T_c\mathcal{C} \to T_{q_i(c)}\mathcal{B}$, manuscript :3073), so distinct sections produce distinct base metrics even from a single fixed fiber metric: $\sigma_i^* g_{\mathcal{B}} \neq \sigma_j^* g_{\mathcal{B}}$ whenever $d\sigma_i \neq d\sigma_j$. There is no contradiction in "fixed fiber metric, agent-relative base geometry" — the agent-relativity enters entirely through the section, which is the agent's belief/frame. The fixedness of $g_{\mathcal{B}}$ is not a hidden global metric on $\mathcal{C}$; a fixed fiber metric does not descend to a canonical base metric, because there is no canonical section.

Second, the fixed fiber metric being shared is what makes the comparison $\Omega_{ij}$ meaningful (it is the same Fisher-Rao geometry both agents' beliefs live in), while contributing no privileged geometry on $\mathcal{C}$. This is the same structural point as perspectival realism's "perspective-independent states of affairs": $g_{\mathcal{B}}$ is the shared fiber-side structure of belief space (Cencov-unique, intrinsic — see info-geometer memo), and $\mathcal{C}$ carries no metric of its own. The manuscript's claim that "all such structure is induced by agent beliefs through the pullback" (:3098) is literally the statement that base geometry $= \sigma^* g_{\mathcal{B}}$ with $\sigma$ the agent's belief — geometrically exact.

## The honest concession
A pedantic version of the objection has a grain: the fiber metric being *fixed and Cencov-unique* means belief-space geometry is not itself agent-relative — only its pullback to $\mathcal{C}$ is. So "no view from nowhere" is a claim about the *base* manifold $\mathcal{C}$ (no privileged metric there), not about belief space (which has the canonical Fisher metric). The manuscript's "no view from nowhere" prose is about $\mathcal{C}$ (:3098 explicitly: "the true state of $\mathcal{C}$"), so this is consistent — but a careful reader should note the scope: there *is* a privileged geometry on the belief fiber; there is none on $\mathcal{C}$.

## External citation
- [Lee2013, pullback of tensor fields]; [doCarmo1992 §1–2]: $G = \sigma^* g$ is the standard metric pullback; distinct sections yield distinct base metrics from one fixed fiber metric; a fixed fiber metric does not induce a canonical base metric absent a canonical section.

## Falsification condition (argued unmet)
The claim fails (internal contradiction) if "$\mathcal{C}$ has no intrinsic metric" requires the fiber metric to also be agent-relative, or if a fixed fiber metric forced a canonical base metric. Standard pullback geometry shows neither: $\sigma_i^* g_{\mathcal{B}}$ is section-dependent, so a fixed $g_{\mathcal{B}}$ is consistent with no canonical metric on $\mathcal{C}$. Condition unmet.
