## Prune to frozen-legacy: delete `vfe/`, `aif/`, `pure_vfe/`

The living `transformer/vfe/` implementation (and the `aif/` active-inference
package that rides on it) has been moved to a separate standalone repository.

Branch: `chore/strip-vfe-aif-pure-vfe-legacy-2026-05-25` off the local HEAD
(the local repo is the source of truth; `main` was 316 commits / 3 weeks stale
and predated `aif/`, so it was not used as the base).

functional imports of `vfe`/`aif`/`pure_vfe`; the dependency layering is
one-directional (`core` never imports the deleted packages). The only
keep-set coupling was in training infrastructure and one mixed test file.
